Reading Note Template

Copy this structure to start a new note for a paper, book or technical report.

Basic Information

Reference: authors, year, title, conference or journal.
Links: PDF, code, data or project page.

Core Question

What question does this work try to answer? Why does the question matter? How is it connected to my research?

Method and Evidence

Record the method, experimental design, data sources, key assumptions and main evidence. Keep the authors' claims separate from my own interpretation.

Reusable Ideas

Write down concepts, metrics, figures, experimental settings or writing patterns that could transfer to my own work.

Follow-Up Questions

List what is still unclear, related papers to check next, and questions that could grow into a project or paper.
