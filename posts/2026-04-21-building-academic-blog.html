从 GitHub Pages 开始搭建个人学术博客

一个个人学术博客不必复杂。最重要的是它能稳定访问、结构清楚，并且让论文、项目和笔记自然沉淀下来。

这个站点包含什么

当前版本采用纯静态文件，不依赖构建工具。主页放置个人简介、研究方向、论文列表、近期文章、简历时间线和联系方式；博客页用于集中展示研究札记。

推荐的维护方式

更新论文时优先写清楚标题、作者、年份、发表地点和链接。写博客时，建议一篇文章只围绕一个问题展开，记录动机、材料、方法、结论和仍未解决的疑问。

部署方式

把这些文件推送到 GitHub 上名为 caotengde.github.io 的仓库后，GitHub Pages 会把主页发布到 https://caotengde.github.io。
